# Imagenes del sitio

Coloca aqui las imagenes publicas del proyecto.

Para la pagina de bienvenida, agrega una imagen con este nombre:

```text
public/images/bienvenida-video.jpg
```

Esa imagen se mostrara en la primera pagina como portada del video introductorio.

Tambien puedes agregar futuras imagenes con nombres descriptivos, por ejemplo:

```text
public/images/evaluacion.jpg
public/images/diagnostico.jpg
public/images/tratamiento.jpg
public/images/prevencion.jpg
```

En React/Vite se usan asi:

```jsx
<img src="/images/bienvenida-video.jpg" alt="Descripcion" />
```
