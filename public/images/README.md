# Imágenes y videos del sitio

Coloca aquí las imágenes públicas del proyecto. PsicoAlma debe priorizar imágenes cálidas: manos, luz natural, naturaleza, espacios tranquilos, ilustraciones amables y personas con expresión serena.

Evita usar imágenes clínicas como portada principal. Si se usan imágenes hospitalarias, que sea en secciones de contexto y con texto cuidadoso.

## Recomendación para nuevos videos

```text
Duración: 8 a 20 segundos
Formato: .mp4
Peso ideal: menos de 8 MB
Estilo: naturaleza, manos, luz, respiración, calma
Audio: opcional, pero debe iniciar silenciado en la página
```
